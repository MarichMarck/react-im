## 简介
react + antd + redu  仿微信网页版，纯前端项目，未对接后端，数据纯前端模拟

## 技术选型

- react
- redu 
- antd （3. ）

## 安装 

```shell
git clone https://github.com/MarichMarck/react-im.git
cd react-im
yarn install  
yarn start
```

## 功能

-[ ] 会话列表  
-[ ] 会话窗口  
-[ ] 通讯录  
-[ ] 设置  
-[ ] 全屏切换
